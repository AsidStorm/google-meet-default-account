Auth User
What is an AuthUser ID?

To find the AuthUser ID for the Google account you want to set as default:

  • Go to meet.google.com
  • Using the account switcher in the top right, switch to the google account you want as your default
  • After the page reloads with the desired account active, there will be a authuser= section added to the url. The number that follows this is your AuthUser ID

If the first account that Google Meet opened is your desired account, your AuthUser ID is 0 (and you probably don't need this extension)

What is an Force Redirect?

This option is required in order to handle invalid links with incorrect AuthUser inside it.

This option start to redirect you into AuthUser ID, no matter which AuthUse ID provided in URL.

What is an Hide AuthUser after redirection?

This option helps you to remove ?authuser= after redirection from url, so you can easily share it to others by copy/pasting.
